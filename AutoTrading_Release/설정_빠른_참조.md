## 🔥 자주 변경하는 설정 Top 5

### 1️⃣ 거래 모드 (모의투자 ↔ 실전투자)

**파일**: `.env` (5번째 줄)

```env
# 모의투자 (안전)
USE_PAPER_TRADING=True

# 실전투자 (실제 거래)
USE_PAPER_TRADING=False
```

**변경 후**: 프로그램 재시작 필요

## 📊 전체 설정 치트시트

### .env 파일

```env
# API 인증 정보
KIS_APP_KEY=your_app_key_here
KIS_APP_SECRET=your_app_secret_here
KIS_ACCOUNT_NUMBER=12345678-01

# 거래 모드
USE_PAPER_TRADING=True  # True=모의투자, False=실전투자
```

---

### config.py (공통 설정)

```python
# API 인증 (환경변수에서 로드)
KIS_APP_KEY = os.getenv("KIS_APP_KEY")
KIS_APP_SECRET = os.getenv("KIS_APP_SECRET")
KIS_ACCOUNT_NUMBER = os.getenv("KIS_ACCOUNT_NUMBER")
USE_PAPER_TRADING = os.getenv("USE_PAPER_TRADING", "True") == "True"

# 매매 스케줄
SELL_INTERVAL_MINUTES = 30  # 매도 체크 주기 (분)
BUY_INTERVAL_MINUTES = 60   # 매수 체크 주기 (분)

# 목표 수익률 (공통)
PROFIT_THRESHOLD = 0.05  # 5%

# 재시도 설정
MAX_RETRY_COUNT = 3
ORDER_TIMEOUT_MINUTES = 5

# 로그 설정
LOG_LEVEL = "INFO"
LOG_FILE = "trading.log"
```

---


## 🎯 시나리오별 빠른 설정

### 시나리오 1: 공격적 트레이딩 (단기 매매)

```python
# config.py
PROFIT_THRESHOLD = 0.03  # 3% 익절

# kr/config.py
STOP_LOSS_THRESHOLD = -0.05       # -5% 손절
STOP_LOSS_COOLDOWN_DAYS = 7       # 7일 재매수 금지

# us/config.py
STOP_LOSS_THRESHOLD = -0.07       # -7% 손절
STOP_LOSS_COOLDOWN_DAYS = 1       # 1일 재매수 금지
```

**특징**: 빠른 진입/청산, 짧은 보유 기간

---

### 시나리오 2: 보수적 트레이딩 (중장기 매매)

```python
# config.py
PROFIT_THRESHOLD = 0.10  # 10% 익절

# kr/config.py
STOP_LOSS_THRESHOLD = -0.15       # -15% 손절
STOP_LOSS_COOLDOWN_DAYS = 90      # 90일 재매수 금지

# us/config.py
STOP_LOSS_THRESHOLD = -0.20       # -20% 손절
STOP_LOSS_COOLDOWN_DAYS = 7       # 7일 재매수 금지
```

**특징**: 느긋한 진입/청산, 긴 보유 기간

---

### 시나리오 3: 균형 트레이딩 (기본값)

```python

# us/config.py

```

**특징**: 중간 수준의 리스크/리턴

## ⚠️ 주의사항

### 변경하면 안 되는 설정

❌ **절대 변경 금지**:
- API 엔드포인트 URL
- 토큰 파일 경로
- 거래소 코드

❌ **신중히 변경**:
- 계좌번호 (잘못 입력 시 거래 실패)
- API 키 (잘못 입력 시 인증 실패)
