## 📝 고객에게 제공할 초기 설정 가이드

### 1단계: 압축 해제
```
AutoTrading_v4.7_Release.zip을 원하는 위치에 압축 해제
예: C:\AutoTrading\
```

### 2단계: Python 설치 확인
```
Python 3.8 이상 설치 필요
https://www.python.org/downloads/
```

### 3단계: 자동 설치
```
setup.bat 더블클릭
→ 모든 패키지 자동 설치
→ .env 파일 자동 생성
```

### 4단계: API 키 설정
```
.env 파일 열기 (setup.bat 실행 시 자동으로 열림)

다음 정보 입력:
KIS_APP_KEY=고객의_APP_KEY
KIS_APP_SECRET=고객의_APP_SECRET
KIS_ACCOUNT_NUMBER=고객의_계좌번호
USE_PAPER_TRADING=True  # 처음엔 모의투자!
```

### 5단계: 실행
```
start.bat 더블클릭
→ 자동매매 시스템 시작
```

## 🔧 기술 지원 준비사항

### 자주 묻는 질문 (FAQ)

**Q1. Python이 없는데요?**
```
A: https://www.python.org/downloads/ 에서 설치
   "Add Python to PATH" 옵션 필수 체크!
```

**Q2. API 키는 어디서 발급받나요?**
```
A: https://apiportal.koreainvestment.com
   회원가입 → 앱 등록 → 키 발급
```

**Q3. 모의투자는 어떻게 하나요?**
```
A: .env 파일에서
   USE_PAPER_TRADING=True 설정
```

**Q4. 실전 투자는 언제 시작하나요?**
```
A: 최소 1주일 이상 모의투자 테스트 후
   USE_PAPER_TRADING=False로 변경
```

**Q5. 에러가 발생하면?**
```
A: trading.log 파일 확인
   USER_MANUAL.md의 "문제 해결" 섹션 참조
```
